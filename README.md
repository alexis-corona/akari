# AKARI (あかり)

Web-based platform to learn programming with Java.

## Website

Visit the tool here: [https://akari-code.web.app](https://akari-code.web.app)

## Description

A web platform built with **HTML, CSS, and Vanilla JavaScript** designed to teach programming concepts cleanly, stripping away unnecessary complexities. It guides students through a structured curriculum—from syntax fundamentals to advanced design patterns and architecture.

## Features

- **Java Master Roadmap:** Sequentially structured into 6 logical milestones (Syntax, OOP, Collections, Algorithms, Design Patterns, and Architecture).
- **Dark-Themed UI:** Minimal, sleek dashboard inspired by modern Gemini ergonomics to reduce eye strain during long coding sessions.
- **Accessible Content:** All learning guides, syllabi, and modular card details are written in Spanish to eliminate language barriers for beginner developers.
- **Production Ready:** Optimized and lightweight codebase deployed via modern web workflows.

## Tech Stack

- **Languages:** HTML5, CSS3, JavaScript (ES6+)
- **Architecture:** Semantic HTML markup and responsive CSS Grid/Flexbox layouts.
- **Analytics:** Integrated with Google Analytics via Firebase SDK for tracking student engagement.
- **Hosting:** Firebase
